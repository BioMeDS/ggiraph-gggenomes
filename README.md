# ggiraph-gggenomes
Demo for basic interactivity on gggenomes plots with ggiraph.

The demo itself lives in [`index.Rmd`](index.Rmd), which knits to
`index.html`. Because the demo relies on ggiraph's interactive SVG/JS output
(tooltips, hover, zoom), it needs to be viewed as rendered HTML rather than
as a GitHub-rendered Markdown file — e.g. via GitHub Pages once enabled for
this repo, or by knitting `index.Rmd` locally and opening `index.html` in a
browser.
